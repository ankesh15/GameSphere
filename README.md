# GameSphere Monorepo

AI-powered gamer community, matchmaking, and game discovery platform.

## Structure
- `frontend/`: React + TypeScript + Vite client
- `backend/`: NestJS + TypeScript REST API
- `ai-service/`: FastAPI service for AI scoring
- `shared/`: Shared types and constants

## Getting Started
Each service is independently runnable. See the README in each folder for
setup and dev instructions.
